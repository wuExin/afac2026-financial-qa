# AFAC2026 赛道四：赛事介绍

> 来源：[天池大赛赛制页](https://tianchi.aliyun.com/competition/entrance/532486)
> 赛题全称：AI大模型赛 AFAC2026挑战组-赛题四：金融长文本Agent 的动态记忆压缩与高效问答挑战

---

## 一、参赛规则

1. 面向全球开放，高校、科研单位、企业等人员均可通过天池平台报名。选手仅可在挑战组、初创组中选择一个组别报名，报名挑战组的选手可选择多个命题参赛。
2. 选手可单人成队或最多不超过3人组队参赛，每位选手只能加入一支队伍，在挑战组范围内，每队可选多个赛题参赛。
3. 为保障参赛队伍成员信息有效性，所有选手需完成实名认证，未完成不可进入B榜和决赛路演。
4. 参赛队伍应按要求提供所有必要信息，必要时应按大赛组委会要求补充提供。
5. 大赛组委会成员、专家评委及其研究团队，以及其他有机会接触赛题背景的相关人员，无参赛资格。蚂蚁集团（含集团内子公司）员工，可参赛排名，但不参与评奖及领取奖金。

---

## 二、比赛流程

比赛流程分为报名认证、代码提交、报告提交、决赛路演等阶段。

### 阶段一：报名认证

**时间**：2026年6月8日10:00（周一）- 7月20日12:00（周一）

1. 登录赛题页，完成个人信息注册，即可报名参赛。
2. 选手可单人成队或最多不超过3人组队参赛，每位选手只能加入一支队伍。
3. 选手需确保报名信息准确有效，组委会有权取消不符合条件队伍的参赛资格及奖励。
4. **实名认证**：所有队员需在2026年7月20日中午12:00前完成实名认证（认证入口：天池官网-右上角个人中心-认证-支付宝实名认证，或访问 https://tianchi.aliyun.com/home/identification），未完成认证的参赛队伍将被取消比赛资格。

### 阶段二：代码提交

**时间**：2026年6月8日（周一）- 7月24日（周五）

1. 报名成功后，参赛队伍通过天池平台下载训练数据，调试模型与代码，在线提交结果。
2. 6月8日10:00起系统实时评测，每小时更新排行榜，**一天内有3次提交机会**，按照评测指标排序。排行榜展示本阶段的最优成绩。
3. 比赛于7月24日17:00结束。

**A/B 榜时间**：
- A榜数据评测：2026年6月8日10:00 - 7月21日20:00
- B榜数据评测：2026年7月22日0:00 - 7月24日17:00

### 阶段三：报告提交

**时间**：2026年7月底 - 8月5日（周三）

1. 成绩排名前列，并由组委会通知进入评审环节的队伍，需按规定提交团队资料、比赛代码与技术报告。未按时提交或者审查过程中发现有违反比赛规则的队伍将被取消评审和获奖资格。如有队伍被取消评审和获奖资格，将根据天池平台榜单成绩排名递补。
2. 报告提交截止时间：2026年8月5日（周三）23:59。
3. 报告提交时间及方式将于7月底通知。

### 阶段四：决赛路演

**时间**：2026年8月中旬（拟）

1. 决赛采用现场路演方式，对参赛作品进行评审，参赛团队抽签确定路演顺序。
2. 路演答辩包含作品陈述和专家问答，评委将根据选手的技术思路、理论深度和答辩表现进行综合评分，按决赛评分决定最终排名。
3. 决赛路演现场，组委会届时将向入围选手公示在线评测与路演答辩评分权重。
4. 如选手未到决赛路演现场参与答辩，则默认自动放弃决赛资格及奖金。

大赛将对最终排名和队伍进行公示，并拟于9月-11月进行颁奖及奖金发放。

## 五、特别说明

1. 参赛者在分析数据和构建模型时，需遵循国家法律法规、行业规定和道德规范，不得侵犯他人知识产权、隐私权等；
2. 参赛者在比赛过程中，建议与其他参赛者积极互动，分享经验和心得，共同提高参赛水平；
3. 组委会在比赛结束后，经参赛者同意，可将优秀作品和解决方案分享至大赛指定宣发平台；
4. 参赛者须严格遵守比赛规则与公平竞赛原则，严禁以任何形式弄虚作假或作弊。一经查实存在违规行为，大赛组委会有权立即取消其参赛资格及成绩，并全额追回已发放的奖金及相关奖励；
5. 大赛组委会保留对本方案的最终解释权，并有权根据实际情况调整竞赛流程和相关要求。
